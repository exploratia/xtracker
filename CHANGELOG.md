# Changelog xTracker

## [1.0.1] - 2025-08-01

### Features

- add tooltips to all pressable icons (on long touch)

First published version

### Features

- daily check series
- blood pressure series
- import, export and share series in JSON format
- series views: table, chart, dots
- dark and light theme support
